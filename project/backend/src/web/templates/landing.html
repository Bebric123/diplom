Error Monitor

Сбор ошибок и логов из приложений, уведомления в Telegram, сводки и отчёты. Зарегистрируйте проект и подключите SDK за несколько минут.

Подключить проект Инструкции и примеры OpenAPI

Регистрация

Форма /register — project_id, API-ключ и привязка Telegram-чата для алертов.

SDK

Python, Node.js, PHP и сырой HTTP: события /track и загрузка логов /logs/upload.

Состояние API

/health — проверка доступности коллектора и БД.

Исходный код и установка из GitHub — см. репозиторий.
