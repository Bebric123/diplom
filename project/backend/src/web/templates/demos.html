How to use (install & run)

  • Коллектор: Docker — project/docker (docker compose up); локально — uvicorn из project/backend (см. README).
  • Регистрация проекта: /register → сохраните UUID и ключ.
  • Telegram: бот в группе и chat id — /docs/telegram.
  • SDK: пошагово — /docs/sdk (Python / Node / PHP, CORS для браузера).
  • Демо из клона: git clone … → каталог examples/sdk-demos; переменные MONITOR_URL, MONITOR_PROJECT_ID, MONITOR_API_KEY.
  • Collector: Docker in project/docker or local uvicorn in project/backend.
  • Project: /register → keep UUID + API key.
  • Telegram: /docs/telegram — bot in group, chat id.
  • SDK guide: /docs/sdk.
  • Runnable demos: clone repo → examples/sdk-demos; env MONITOR_*.
