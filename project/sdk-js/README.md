# error-monitor-sdk (JavaScript / Node.js)

Клиент для коллектора Error Monitor: события и ошибки (`/track`), загрузка логов (`/logs/upload`), интеграции Express / Fastify / браузер.

## Установка

Из клона:

```bash
npm install ./project/sdk-js
```

Или укажите в `package.json` зависимость `file:../../diplom/project/sdk-js` (путь от вашего приложения).

## Документация

Страница на коллекторе: `/docs/sdk`. Демо: `examples/sdk-demos/nodejs/`, `examples/sdk-demos/frontend/`.
